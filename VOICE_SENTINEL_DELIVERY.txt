🎯 KEY FEATURES
═══════════════════════════════════════════════════════════════════════════════

✨ 7 Event Types Supported
   • FALL_DETECTED       [CRITICAL] — Person fell/lying down
   • ABANDONED_OBJECT    [HIGH]     — Unattended bag/package
   • SHADOW_DETECTED     [MEDIUM]   — Movement in dark area
   • LOITERING           [MEDIUM]   — Person in restricted zone
   • WEAPON              [CRITICAL] — Possible weapon detected
   • UPI_FRAUD           [HIGH]     — High-risk transaction
   • CROSS_DOMAIN        [CRITICAL] — Physical + digital threat

✨ 4 REST API Endpoints
   • POST   /api/voice/alert          — Generate alert from event
   • GET    /api/voice/alert/test     — Test with sample event
   • POST   /api/voice/alert/preview  — Preview alert (no call)
   • GET    /api/voice/events/templates  — Get example templates

✨ Professional Voice Alerts
   • Messages under 60 seconds ⏱️
   • Professional, calm tone 🎤
   • No small talk or AI mentions 🔇
   • Event-specific messaging 🎯

✨ DTMF Keypad Menu
   • CRITICAL/HIGH risk: Full menu (press 1 or 2) ☎️
   • MEDIUM risk: No input required 📢

✨ Production Ready

    ↓
